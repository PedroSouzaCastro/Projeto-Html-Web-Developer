Ferramentas ultilizadas e Estruturas de Sites.

  • VOLTAR

Para criar nosso sites e projetos, antes de tudo precisamos ter a base para fazer isso acontecer. A base que estou dizendo é o software que vai fazer nossos projetos e sites futuros. primeiro programa é o Visual Studio Code, editor de código-fonte criado pela Microsoft para Windows, MacOS,Linux.
Nele você pode usar varias linguagens de programação por exemplo JavaScript, Python e Java, até as mais especializadas, como Go, Rust e TypeScript.
Na estrutura do html você sempre vai ter a parte "head" do site, que significa a cabeça de seu projeto. Nele vai ficar o titulo do site, o idioma de seu site, informações de estilo, e tudo aquilo que não é visível para o visitante do site. Diferente do "body" que significa o corpo do seu site. Nele vai ficar todo o conteúdo de seu projeto por exemplo, imagens, textos, conteúdo, links entre outros, tudo que é visível para o visitante.

Tags HTML para TEXTOS

  • VOLTAR

Toda tag de marcação do HTML, tem seus significados e comandos. Por exemplo para colocar um cabeçalho em seu site tem a tag própria para isso, essa tag é a <h1>. É os cabeçalhos tem diversos tamanhos e níveis para sua escrita, <h2>, <h3>, <h4>, <h5> e o <h6>.
Todo texto tem seu cabeçalho, e também temos o parágrafo. No HTML a tag do Parágrafo e a <p>, nela você vai escrever seu conteúdo.
Além da parte textual tem as tags funcionais, por exemplo a <hr> vai criar uma linha horizontal para seu site, a tag <i> vai deixar seu texto itálico,Tag <u> para deixar sublinhado assim.

O que é lista ordenadas e não ordenadas?

  • VOLTAR

Para colocar listas em seus códigos são bem simples, mas antes de saber a tag precisamos aprender cada uma delas. A lista ordenadas significa que sempre haverá uma ordem nela dependendo da ordem específica, como numérica ou alfabética. Para enumerar elementos de seu projeto ela e o essencial, para fazer o código html basta colocar <ol> e dentro dela <li>. Assim enumerando tudo que você coloca dentro de <li>.
Para colocar o tipo alfabética ou romano: <ol type="A, I,">. "A" para ordem alfabética maiúscula, "I" para odem maiúscula em Romano. Para ser minúsculas basta colocar letras minúsculas.
Diferente é ao contrário das listas ordenadas, as listas não ordenadas não tem uma ordem distinta, elas apenas coloca marcadores dentro de cada elemento. Assim ficando <ul> e dentro dela <li>. Desse modo criando um tipo de tópicos, que pode ser usado para diversas coisas na criação de sites e na programação.

Links para navegação.

  • VOLTAR

Uma tag essencial em qualquer projeto e site que criarmos hoje, tem que ter uma navegação apropriada dentro dele. Seja para ir em algum site ou ir há outro conteúdo criado por você. Para usar a "navegação" de seu site, existe o código com a tag <a> que significa "Âncora", nela é possível ancorar de um site há outro. A tag completa é <a href="Link para navegação">nome do link</a>. As palavras entre as tags de abertura e fechamento da âncora serão exibidas como um hipervínculo.
Link para uma página externa (outro site): <a href="https://www.exemplo.com">Visitar Exemplo.com</a>.
Link para uma página interna (dentro do seu próprio site): <a href="pagina.html">Ir para a Página</a>.
Link para um endereço de e-mail: <a href="mailto:seuemail@example.com">Enviar um e-mail</a>.
Link para um arquivo para download: <a href="documento.pdf" download>Baixar o Documento PDF</a>.
Link para um número de telefone (para dispositivos móveis): <a href="tel:+1234567890">Ligar para +123-456-7890</a>.
